# LibreSpeed

[LibreSpeed](https://librespeed.org/) is a FOSS, self-hostable speedtest server available on [Github](https://github.com/librespeed/speedtest). We run a [podman container](https://github.com/librespeed/speedtest/pkgs/container/speedtest) of it on [fw-mesh-host](../networking/servers/fw-mesh-host.md).
