## Overview

Often, we will notice a node go down in UISP and learn that this is because the current resident has moved without letting us know. The goal of going on site is to introduce our project to any new residents and get them connected if they want. If they don't want to get connected, remove the existing hardware.

## Materials needed

- Drill and/or impact driver
- Screwdriver
- Socket set
- Ladder
- OmniTIK AC adapter
- Inside router
- Node information card
- Phone and/or laptop
- Postcard or paper
- Pen or pencil

## Try to make contact with previous node contact

Send a message to the contact for the node listed in the Trello card using all available contact methods. If you're able to make contact with them, confirm that they've moved. Ask if they can share contact information for remaining or current residents.

## Visit home and talk with resident or leave a note

If you have contact information for people still living at the node's address, reach out to them and determine if they want to get reconnected to Tucson Mesh. Schedule a time to get them reconnected or to deinstall the equipment.

If you can't contact the current resident ahead of time, visit the node and knock on the door. Explain Tucson Mesh and answer any questions. If you have the time and equipment, and they want to be reconnected, you can proceed with restoring power to the Omni and installing a new indoor router.

If they need more time to think, be sure to get contact information for them.

If no one answers, leave a note (or a fun postcard!) with contact information. A sample note could be something like:

> Did you know you have pay-what-you-can, community-powered internet on your roof? A previous resident connected this home to Tucson Mesh and we'd love to get you reconnected. You can email us at support@tucsonmesh.net or text/call {phone_number} to schedule an appointment! Hope to talk to you soon, {name}

## Reconfigure or install inside router

You may need to replace the AC adapter for the OmniTIK because it was accidentally thrown away or taken when the previous resident moved.

Fill out a node information card so that the resident knows how to get support and contact Tucson Mesh.

Tell the resident about how to make a donation and suggest a donation amount.

Help them install Slack and add them to the Tucson Mesh Slack.

## Update node record

If the node was reconnected, leave a comment describing the steps taken to get the connection restored. Be sure to update the contact information to the new residents.

If you left a note, leave a comment mentioning that you did this.

If you made contact, but the current residents wanted time to decide if they wanted to connect, update the contact information in the card.

If the hardware was removed, move the card to the `Uninstalled` list.
